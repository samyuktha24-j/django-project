CyberSecurity Management System (CSMS)

A centralized platform designed to manage cryptographic operations, enforce security policies, and ensure data confidentiality, integrity, and authenticity using Public Key Infrastructure (PKI).

Purpose of CSMS

The CyberSecurity Management System is developed to demonstrate the practical implementation of cryptographic mechanisms used in secure software systems.

Core Capabilities

Key Generation

Secure AES and RSA key creation.

Encryption

Confidential file encryption and decryption.

Digital Signature

Authenticity and integrity verification.

Signature Validation

Detection of file tampering.
